# Android-Camera
## Bilgi

DJI SDK kullanılarak dron üzerindeki kamerayı kontrol etmek  **Fotograf Çekimi** and **Video Kaydı**.
AndroidManifest.xml dosyasındaki KEY için  lütfen DJI sitesine üye olarak Developer KEy almayı unutmayınız.
## Gereksinimler

 - Android Studio 2.0+
 - Android System 4.1+
 - DJI Android SDK 4.9
